1.安装gdb
    ```
    sudo apt install gdb
    ```
2.gdb常用命令
    如果希望程序可以调试，编译时需要加入-g选项，并且不能使用-o优化选项。

    ```
    set args
    ```
    设置程序运行的参数。
    eg ```set args 张三 西施 你好鸟```


    ```break```简写```b```
    设置断点，```b 20```表示在20行设置断点，可以设置多个断点.


    ```run```简写```r```
    运行程序，遇到断点停下来


    ```next```简写```n```
    执行当前语句，若当前语句为函数调用，不会进入函数内部


    ```step```简写```s```
    执行当前语句，若当前语句为函数调用，会进入函数内部
    但是若函数时库函数或第三方函数，s也进不去，因为没有源码


    ```continue```简写```c```
    继续运行程序，遇到下一个断点停止，如果没有则一直运行


    ```print```简写```p```
    eg:```p name1```
    显示变量或表达式的值，如果p后面是表达式，会执行这个表达式
    eg:```p (size_t)strlen(message)```因为gdb不知道表达式的返回值，所以需要显示强制类型转换为(size_t)
    p命令也可以改变变量的值
    eg：```p ii=8```
    



    ```set var```
    设置变量的值
    eg：```set var ii = 11```
        ```set var name = "西施"```


    ```quit```简写```q```
    退出gdb
